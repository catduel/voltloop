# Bury It!

`Bury It!` is a fully anonymous text-first app for digitally burying memories, notes, chat fragments, and emotional clutter that users do not want to keep in sight but are not ready to delete forever.

This workspace contains the initial product design package for a visual-free V1.

## Files

- `docs/product-overview.md`: Brand, positioning, value proposition, and product concept.
- `docs/v1-prd.md`: Core product requirements, user stories, and feature scope.
- `docs/information-architecture.md`: App structure, navigation, data model, and key flows.
- `docs/screen-copy.md`: Full English copy for the main screens.
- `docs/safety-and-anonymity.md`: Rules for anonymity, moderation, and trust.

## Product Direction

- No image uploads in V1
- No real names
- Fully anonymous participation
- Text-first emotional release
- Optional public graveyard with strict masking and moderation
